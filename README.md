welcome to the untalented game.

Controls:
W/A S/D = to move
Shift + (to move) = sprint
E = spawn object 
Right Click = grab / throw
--------------------------
grab = 1
throw = 2
Right Click = rmb 

if rmb held; make grab
if rmb released; make throw
--------------------------
1,2,3 = switch tools 
tools = cube, ball, NPC
Tab = slow motion
V = noclip
F11 = fullscreen
